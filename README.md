Car OBD multi gauge for ESP32
TFT Adafruit GC9A01A

Features:
* Multi-thread / Dual core
* Graphical display
* Low/High values for every gauge (with warning colors)
* Dual gauge display
* Up to two TFT displays
* User settings (saved in EEPROM)
